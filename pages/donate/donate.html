¿Cómo puedo ayudar?

Puedes ayudar de muchas formas, desde donar dinero hasta donar tu tiempo.

Donar dinero

Puedes donar dinero para ayudar a los animales a tener una mejor calidad de vida. El dinero se utiliza para comprar comida, medicinas y pagar los servicios veterinarios.

Donar tu tiempo

Puedes donar tu tiempo para ayudar a los animales a tener una mejor calidad de vida. El tiempo se utiliza para cuidar a los animales, limpiar el refugio y darles cariño.
